# aws-auto-scaling-project
Designed and deployed a highly available and fault-tolerant web application on AWS. The application uses EC2 instances with Nginx hosting a login page, an Application Load Balancer for traffic distribution, and an Auto Scaling Group to automatically adjust instances based on CPU utilization across multiple Availability Zones. 

 # AWS Auto Scaling Web Application

This project demonstrates a highly available and fault-tolerant web architecture on AWS.

## Architecture
- EC2 instances
- Application Load Balancer
- Target Group
- Auto Scaling Group
- Multi Availability Zone deployment

## Features
- Self-healing infrastructure
- Automatic scaling based on CPU
- High availability

## Tools Used
- AWS EC2
- AWS ALB
- AWS Auto Scaling
- Nginx
